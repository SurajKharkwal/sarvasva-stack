# SARVASVA Helper CLI

A helper CLI for the main CLI [@sarvasva-app/init](https://www.npmjs.com/package/@sarvasva-app/init).

It generates a complete React(vite) starter app with the following features:

- Tanstack-Router
- Authentication
- Database
- UI library
- ORM
